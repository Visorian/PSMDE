# Clear-MdeAuthorizationInfo

## SYNOPSIS
Clears the authorization information that is used to get a valid MDE token.

## SYNTAX

```
Clear-MdeAuthorizationInfo [<CommonParameters>]
```

## DESCRIPTION
Clears the authorization information that is used to get a valid MDE token.

## EXAMPLES

### EXAMPLE 1
```
Clear-MdeAuthorizationInfo
```
