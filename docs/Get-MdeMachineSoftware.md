# Get-MdeMachineSoftware

## SYNOPSIS
Retrieves a collection of installed software related to a given device ID.

## SYNTAX

```
Get-MdeMachineSoftware [-id] <String> [<CommonParameters>]
```

## DESCRIPTION
Retrieves a collection of installed software related to a given device ID.

## EXAMPLES

### EXAMPLE 1
```
Get-MdeMachineSoftware -id '123'
```
